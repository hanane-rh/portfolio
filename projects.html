Project · 01

GED — Documentation Management System

Spring BootReactPostgreSQL

Client

Sonatrach — Built as a final-year thesis project (Mémoire de Fin d'Études) for one of Algeria's largest enterprises.

Overview

An enterprise document management system (GED) designed to handle the full documentation lifecycle within a large corporation like Sonatrach. The system provides document classification, codification, archiving, digital signature workflows, advanced search capabilities, audit trails, and traceability — ensuring structured, secure, and compliant document management across departments.

System Actors

The system involves three main actors: the Admin who manages users and global configuration; the External Client (a company or individual seeking to collaborate with Sonatrach); and the DCD (an intermediary between external clients and the enterprise), as well as the Department Head who oversees document workflows within their unit.

My Role & Team

My Role Full Stack Developer (Backend + Frontend) I contributed to both backend development with Spring Boot and frontend implementation with React — covering API design, authentication, document workflows, and UI integration.

Key Features

  • Document Classification — Structured categorisation and codification of all documents.
  • Archiving — Secure long-term storage with version control and traceability.
  • Digital Signatures — Multi-step signature workflows with role-based approval chains.
  • Advanced Search — Full-text search with filters by category, date, actor, and status.
  • Audit Trail — Complete history of document actions for compliance and accountability.
  • Multi-Actor Access — Role-based interfaces for Admin, External Client, DCD, and Department Head.

Project · 03

Biblioteka

DjangoReact

Context

Built as part of an internal web development challenge.

Overview

Biblioteka is an online bookstore offering a clean, intuitive shopping experience for book lovers. Users can browse a rich catalogue, search and filter by category, add books to their cart, choose between in-store pickup or home delivery, and pay via multiple methods. The backend is powered by Django REST Framework for secure authentication and order management, consumed by a React frontend.

My Role & Team

My Role Backend Developer (100%) I was solely responsible for the entire backend: user authentication (JWT), payment integration, delivery system, order management, and the REST API consumed by the frontend.

  • Shopping Cart — Smooth cart management with real-time price calculations.
  • Multiple Payment Options — Credit card, digital wallet, and cash on delivery.
  • Delivery & Pickup — Choose between home delivery or in-store collection.
  • User Accounts — Secure registration, login, order history, and wishlist management.
  • Order Tracking — Real-time delivery status updates for customers.
